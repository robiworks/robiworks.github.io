Komande za IPv4

PC1 (192.168.C.100):

sudo ifconfig eth0 192.168.C.100
sudo ifconfig eth0 netmask 255.255.255.0
sudo ip route add 0.0.0.0/0 via 192.168.C.1 dev eth0

PC2 (172.16.D.100):

sudo ifconfig eth0 172.16.D.100
sudo ifconfig eth0 netmask 255.255.255.0
sudo ip route add 0.0.0.0/0 via 172.16.D.1 dev eth0

R1 (192.168.C.1, 10.A.0.1):

ifconfig eth0 192.168.C.1
ifconfig eth0 netmask 255.255.255.0
ifconfig eth1 up
ifconfig eth1 10.A.0.1
ifconfig eth1 netmask 255.255.255.252
ip route add 172.16.D/24 via 10.A.0.2

R2 (172.16.D.1, 10.0.B.1):

ifconfig eth0 172.16.D.1
ifconfig eth0 netmask 255.255.255.0
ifconfig eth1 up
ifconfig eth1 10.0.B.1
ifconfig eth1 netmask 255.255.255.252
ip route add 192.168.C/24 via 10.0.B.2

R3 (10.A.2, 10.0.B.2):

ifconfig eth0 netmask 255.255.255.252
ifconfig eth1 up
ifconfig eth1 10.0.B.2
ifconfig eth1 netmask 255.255.255.252

Podomrežja IPv6

Tukej morš pazit, ker če sta A ali C v vpisni 0, potem lahko krajšaš naslov (pride :0:: -> ::)!

2001:db8:e:A::/64
2001:db8:B:f::/64
2001:db8:ef:C::/64
2001:db8:D:ef::/64

Komande za IPv6

PC1 (2001:db8:ef:C::2/64):

sudo modprobe ipv6
sudo ip -6 addr add 2001:db8:ef:C::2/64 dev eth0
sudo ip -6 route add ::/0 via 2001:db8:ef:C::1 dev eth0

PC2 (2001:db8:D:ef::2/64):

sudo modprobe ipv6
sudo ip -6 addr add 2001:db8:D:ef::2/64 dev eth0
sudo ip -6 route add ::/0 via 2001:db8:D:ef::1 dev eth0

R1 (2001:db8:ef:C::1/64, 2001:db8:e:A::2/64):

ip -6 addr add 2001:db8:ef:C::1/64 dev eth0
ip -6 addr add 2001:db8:e:A::2/64 dev eth1
ip -6 route add 2001:db8:D:ef::/64 via 2001:db8:e:A::1

R2 (2001:db8:D:ef::1/64, 2001:db8:B:f::2/64):

ip -6 addr add 2001:db8:D:ef::1/64 dev eth0
ip -6 addr add 2001:db8:B:f::2/64 dev eth1
ip -6 route add 2001:db8:ef:C::/64 via 2001:db8:B:f::1

R3 (2001:db8:e:A::1/64, 2001:db8:B:f::1/64):

ip -6 addr add 2001:db8:e:A::1/64 dev eth0
ip -6 addr add 2001:db8:B:f::1/64 dev eth1

ip -6 route add 2001:db8:ef:C::/64 via 2001:db8:e:A::2
